# Alt Tag Bulk Uploader
## Installation
Upload to the wp-content/plugins directory and activate via the CMS - that's it!

## Usage
Navigate to Tools > ATBU Uploader
Click the download button to generate + download a csv file with:
1. ID of the media item
2. Url of the media item
3. Current alt tag of the media item

Edit the file and import via the input - there's no need to upload the CSV file, most of the heavy work is done via the browser
